# Password Strenght Checker
## Description
A beginner cybersecurity project developed in python to evaluate password strenght based on common security rules.

## Feautures
- Check password lenght
- Check uppercase letters
- Check lowercase letters
- Check numbers
- Check special characters
- Display password srenght (weak/medium/strong)

## Technology
python

## How It Works
The program analyzes a password and checks:
1. Length (minimum 8 characters)
2. Uppercase letters
3. Lowercase letters
4. Numbers
5. Special characters



Based on these checks, it classifies the password as Weak, Medium, or Strong.
